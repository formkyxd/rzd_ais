# АИС «Перевозки РЖД»

Автоматизированная информационная система управления грузовыми железнодорожными перевозками.

**Версия:** 1.0.0  
**Сайт:** https://projectrzd.page.gd  
**БД:** MariaDB 11.4 на хостинге InfinityFree

---

## Установка

### 1. Загрузка файлов
Загрузить все файлы проекта на хостинг через FTP или файловый менеджер панели управления.

### 2. База данных
Создать БД через панель управления хостинга (не через SQL).  
Импортировать через phpMyAdmin: **Импорт → выбрать файл** (не копировать в поле запроса).

> ⚠️ При импорте выбирать кодировку **utf-8**, иначе кириллица превратится в `???`.

### 3. Настройка подключения
Отредактировать `config.php`:

```php
define('DB_HOST', 'ваш_хост');
define('DB_NAME', 'имя_базы');
define('DB_USER', 'пользователь');
define('DB_PASS', 'пароль');
define('BASE_URL', 'https://ваш-сайт.com');
```

## Роли пользователей

| Роль | Доступ |
|------|--------|
| `admin` | Полный доступ ко всем разделам |
| `manager` | Отчёты |
| `dispatcher` | Дашборд, заявки, рейсы, маршруты, просмотр клиентов и локомотивов |
| `mechanic` | ТО, локомотивы, смены |
| `client` | Свои заявки и рейсы |
| `gov_rep` | Дашборд, государственные отчёты |

## Таблицы базы данных

| Таблица | Описание |
|---------|----------|
| `roles` | Роли пользователей |
| `users` | Пользователи системы |
| `clients` | Организации-клиенты |

| `locomotives` | Парк локомотивов |

| `trips` | Рейсы |
| `trip_events` | События рейса (отправление, задержки, прибытие) |
| `maintenance_records` | Записи ТО и ремонтов |
| `maintenance_types` | Виды ТО (ТО-3, ТР-1, КР и др.) |
| `invoices` | Счета клиентам |
| `payments` | Платежи по счетам |
| `shifts` | Смены диспетчеров и механиков |
| `audit_log` | Журнал действий пользователей |

## Статусы

**Заявки:** `new` → `processing` → `approved` / `rejected` → `completed` / `cancelled`

**Рейсы:** `planned` → `in_progress` → `completed` / `cancelled` / `delayed`

**Локомотивы:** `operational` / `maintenance` / `repair` / `decommissioned`

**Счета:** `draft` → `issued` → `paid` / `overdue` / `cancelled`
